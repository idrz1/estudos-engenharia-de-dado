# 🐍 O Básico - Python
Repositório dedicado aos meus estudos práticos de Python, focado na construção de uma base sólida para Engenharia de Dados.

## 📂 Arquivos Neste Repositório

* **`python_basico.py`**: Consolidação dos fundamentos da linguagem. Como já trago uma base de lógica de programação e algoritmos, este arquivo foca na adaptação da sintaxe do Python (Manipulação de Dicionários, Loops, Funções e Tipagem Dinâmica).

## 🎯 Objetivo
Documentar a evolução técnica desde a lógica de programação pura até a manipulação avançada de tabelas e construção de pipelines de dados (ETL).
